## Setup

1. Clone the repository:
```bash
git clone https://github.com/rohitkamra/portfolio.git
cd portfolio
```

2. Add your profile image:
```bash
mkdir data
# Add your image as data/image.png
```

3. Run locally:
```bash
python3 -m http.server 8000
# Open http://localhost:8000
```
